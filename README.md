## 📌 Project Overview
**SkyCast** is a modern, cloud-hosted weather application designed to provide real-time, highly accurate meteorological data. With a clean, glassmorphism-inspired UI, users can easily search for global locations and receive comprehensive current conditions alongside a reliable 7-day forecast.

The application fetches live data using the **Open-Meteo API** and is deployed in the cloud for high availability and seamless performance.

## ✨ Key Features
* **Real-Time Data:** Instant access to current temperature, weather conditions, and highs/lows.
* **In-Depth Metrics:** Displays Wind Speed, Humidity, UV Index, and Visibility.
* **7-Day Forecast:** A clear, at-a-glance view of the week's upcoming weather.
* **Dynamic Search:** Search functionality to look up weather for different cities globally.
* **Modern UI/UX:** Clean, responsive design with smooth gradients and readable cards.
